# Maze Game

Browser-based maze game in plain HTML/JS/CSS where the player navigates from start (S) to finish (F) using arrow keys.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
```
maze_game/
├── index.html
├── style.css
└── app.js
```

## Development
No installation required. Open `index.html` in a browser:
```bash
open maze_game/index.html
```

## Key Notes
- No dependencies or build step — fully self-contained static files.
- Player uses arrow keys to navigate the maze.
